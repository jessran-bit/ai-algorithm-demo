# Simple AI Algorithm Demo

A very simple browser-based demo showing how an algorithm scores candidates and how biased rules can change outcomes.

## How to use
Open `index.html` in a browser.

## To publish with GitHub Pages
1. Upload the files to a GitHub repository
2. Go to Settings
3. Click Pages
4. Under Source, choose `Deploy from a branch`
5. Select `main` branch and `/root`
6. Save
7. Wait for the GitHub Pages link
